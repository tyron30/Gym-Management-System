# Gym Management System with QR Code Attendance

## System Overview

The Gym Management System with QR Code Attendance is a comprehensive, modern solution designed to streamline gym operations through digital transformation. This web-based system integrates QR code technology for seamless member check-ins, automated attendance tracking, and efficient member management. The system provides role-based access for administrators and cashiers, ensuring secure and efficient gym operations while delivering an enhanced member experience.

## Core Features

### 1. User Authentication & Role Management
- **Admin Panel**: Full system access including member management, reports, and system configuration
- **Cashier Panel**: Member registration, payment processing, and attendance monitoring
- **Secure Login**: Session-based authentication with role-based access control

### 2. Member Management System
- **Member Registration**: Complete member profile creation with personal details
- **Membership Plans**: Support for various plans (Per Session, Half Month, 1-12 Months, Annual)
- **Student Discount System**: Automatic 20% discount for verified students
- **Member Status Tracking**: Active, Expired, Suspended status management
- **Search & Filter**: Advanced filtering by plan, payment status, and search terms

### 3. QR Code Generation & Management
- **Standard Model 1 QR Codes**: ISO-compliant black-and-white QR codes without logos or gradients
- **Unique QR Token Generation**: Secure 64-character hexadecimal tokens for each member
- **QR Code Storage**: Automatic PNG image generation and storage
- **QR Code Viewing**: Modal display with print and download options
- **Printable QR Cards**: Professional layout with gym branding

### 4. Attendance Tracking System
- **QR Code Scanning**: Real-time attendance recording via QR code scanning
- **Duplicate Prevention**: One check-in per member per day rule enforcement
- **Attendance History**: Complete check-in records with timestamps
- **Real-time Updates**: Live attendance statistics and reporting

### 5. Payment Management
- **Multiple Payment Methods**: Cash, GCash, Maya, Bank Transfer support
- **Receipt Generation**: Automated receipt creation with unique receipt numbers
- **Payment History**: Complete transaction records with member details
- **Reference Number Tracking**: Support for digital payment references

### 6. Reporting & Analytics
- **Dashboard Statistics**: Real-time metrics for members, revenue, and attendance
- **Attendance Reports**: Daily, monthly, half-month, and annual attendance summaries
- **Revenue Analytics**: Monthly and annual revenue tracking
- **Member Analytics**: Active members, plan distribution, and expiry tracking

### 7. System Administration
- **Database Management**: Automated migrations and data integrity
- **Backup & Recovery**: Database export/import capabilities
- **System Configuration**: Configurable settings for gym operations
- **User Management**: Employee account management

- **Backend**: PHP 8.0+ with MVC architecture
- **Database**: MySQL with InnoDB engine
- **Libraries**: PHP QR Code library, Bootstrap Icons, Font Awesome
